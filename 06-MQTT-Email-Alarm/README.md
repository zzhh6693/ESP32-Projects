# ESP32 MQTT邮箱报警装置

## 项目功能
当温湿度超过设定阈值时，通过MQTT协议发送消息，并触发QQ邮箱报警。

## 硬件要求
- ESP32开发板
- DHT11温湿度传感器
- 杜邦线3根
- 局域网环境

## 接线说明
| DHT11引脚 | ESP32引脚 |
|----------|----------|
| VCC      | 3.3V     |
| GND      | GND      |
| DATA     | GPIO5    |

## 使用方法
1. 按照接线说明连接硬件
2. 在代码中修改WiFi信息、MQTT服务器信息和QQ邮箱信息
3. 将代码上传到ESP32开发板
4. 当温湿度超过设定阈值时，会自动发送报警邮件

## 关键技术点
- MQTT协议
- SMTP邮件发送
- DHT11传感器驱动
- 异常检测
