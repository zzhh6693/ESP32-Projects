# ESP32 超声波测距

## 项目功能
使用HC-SR04超声波传感器测量距离，并在串口打印结果。

## 硬件要求
- ESP32开发板
- HC-SR04超声波传感器
- 杜邦线4根

## 接线说明
| HC-SR04引脚 | ESP32引脚 |
|------------|----------|
| VCC        | 5V       |
| GND        | GND      |
| Trig       | GPIO2    |
| Echo       | GPIO4    |

## 使用方法
1. 按照接线说明连接硬件
2. 将代码上传到ESP32开发板
3. 打开串口监视器，波特率设置为115200
4. 串口会每隔1秒打印一次测量的距离值

## 关键技术点
- 超声波测距原理
- GPIO输入输出控制
- 定时器使用
