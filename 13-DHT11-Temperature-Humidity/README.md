# ESP32 DHT11温湿度采集

## 项目功能
使用DHT11温湿度传感器采集环境温湿度数据，并在串口和OLED显示屏上显示。

## 硬件要求
- ESP32开发板
- DHT11温湿度传感器
- 0.96寸I2C OLED显示屏（可选）
- 杜邦线若干

## 接线说明
| DHT11引脚 | ESP32引脚 |
|----------|----------|
| VCC      | 3.3V     |
| GND      | GND      |
| DATA     | GPIO5    |

## 使用方法
1. 按照接线说明连接硬件
2. 将代码上传到ESP32开发板
3. 打开串口监视器，波特率设置为115200
4. 串口会每隔1秒打印一次温湿度数据

## 关键技术点
- DHT11传感器驱动
- 单总线通信
- 数据滤波与处理
