# ESP32 红外遥控控制LED

## 项目功能
解码红外遥控器信号，通过遥控器控制LED灯的亮灭和亮度调节。

## 硬件要求
- ESP32开发板
- 红外接收模块
- 通用红外遥控器
- 杜邦线3根

## 接线说明
| 红外接收模块引脚 | ESP32引脚 |
|----------------|----------|
| VCC            | 3.3V     |
| GND            | GND      |
| OUT            | GPIO15   |

## 使用方法
1. 按照接线说明连接硬件
2. 将代码上传到ESP32开发板
3. 打开串口监视器，波特率设置为115200
4. 按下遥控器上的按键，串口会打印对应的键值
5. 使用遥控器上的按键控制LED灯

## 关键技术点
- 红外遥控编解码
- NEC协议
- PWM亮度调节
