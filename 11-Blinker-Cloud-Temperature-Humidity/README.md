# ESP32 Blinker云平台温湿度上传

## 项目功能
将DHT11温湿度传感器数据上传到Blinker物联网云平台，实现远程监控。

## 硬件要求
- ESP32开发板
- DHT11温湿度传感器
- 杜邦线3根
- 局域网环境

## 接线说明
| DHT11引脚 | ESP32引脚 |
|----------|----------|
| VCC      | 3.3V     |
| GND      | GND      |
| DATA     | GPIO5    |

## 使用方法
1. 按照接线说明连接硬件
2. 在Blinker App中创建设备，获取设备密钥
3. 在代码中修改WiFi信息和设备密钥
4. 将代码上传到ESP32开发板
5. 打开Blinker App，即可查看实时温湿度数据

## 关键技术点
- Blinker物联网云平台
- MQTT协议
- DHT11传感器驱动
- 数据上传与远程监控
