# ESP32 UDP板间通信

## 项目功能
实现两块ESP32开发板之间基于UDP协议的无线消息收发功能。

## 硬件要求
- 2块ESP32开发板
- 2根USB数据线

## 接线说明
无需额外接线，使用ESP32板载WiFi功能。

## 使用方法
1. 将`UDP-Sender`代码上传到第一块ESP32（发送端）
2. 将`UDP-Receiver`代码上传到第二块ESP32（接收端）
3. 确保两块ESP32连接到同一个WiFi网络
4. 打开两个串口监视器，波特率设置为115200
5. 发送端会每隔1秒向接收端发送一条消息，接收端收到后会在串口打印出来

## 关键技术点
- UDP协议的基本原理与应用
- ESP32 WiFi STA模式配置
- 套接字(Socket)编程基础
- 网络数据传输与解析
