# ESP32 人体红外传感器

## 项目功能
使用人体红外传感器检测人体活动，实现自动报警功能。

## 硬件要求
- ESP32开发板
- HC-SR501人体红外传感器
- 杜邦线3根

## 接线说明
| HC-SR501引脚 | ESP32引脚 |
|-------------|----------|
| VCC         | 5V       |
| GND         | GND      |
| OUT         | GPIO14   |

## 使用方法
1. 按照接线说明连接硬件
2. 将代码上传到ESP32开发板
3. 当传感器检测到人体活动时，板载LED灯会点亮
4. 无人活动时，LED灯会自动熄灭

## 关键技术点
- 人体红外检测原理
- GPIO输入检测
- 延时与防抖处理
