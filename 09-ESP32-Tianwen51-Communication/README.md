# ESP32与天问51跨平台通信

## 项目功能
实现ESP32与天问51单片机之间的串口通信和引脚通信，完成跨平台LED点灯。

## 硬件要求
- ESP32开发板
- 天问51单片机开发板
- 杜邦线若干
- USB数据线2根

## 接线说明
### 串口通信
| ESP32引脚 | 天问51引脚 |
|----------|-----------|
| GPIO16   | P3.0(RXD) |
| GPIO17   | P3.1(TXD) |
| GND      | GND       |

### 引脚通信
| ESP32引脚 | 天问51引脚 |
|----------|-----------|
| GPIO2    | P1.0      |
| GND      | GND       |

## 使用方法
1. 按照接线说明连接硬件
2. 将ESP32代码上传到ESP32开发板
3. 将天问51代码上传到天问51开发板
4. 打开两个串口监视器，查看通信过程
5. ESP32发送指令，天问51控制LED灯亮灭

## 关键技术点
- UART串口通信
- 跨平台硬件通信
- 数据协议设计
- GPIO输入输出
