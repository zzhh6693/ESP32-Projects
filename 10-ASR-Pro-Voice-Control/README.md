# ESP32 ASR-Pro语音控制LED

## 项目功能
集成ASR-Pro离线语音识别模块，通过语音指令控制LED灯亮灭。

## 硬件要求
- ESP32开发板
- ASR-Pro语音识别模块
- 杜邦线4根

## 接线说明
| ASR-Pro引脚 | ESP32引脚 |
|------------|----------|
| VCC        | 5V       |
| GND        | GND      |
| TX         | GPIO16   |
| RX         | GPIO17   |

## 使用方法
1. 按照接线说明连接硬件
2. 将代码上传到ESP32开发板
3. 说出语音指令："开灯"、"关灯"
4. LED灯会根据语音指令做出相应的动作

## 关键技术点
- 离线语音识别
- 串口通信
- 语音指令解析
