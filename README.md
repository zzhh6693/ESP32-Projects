# ESP32-Projects
ESP32 单片机学习项目合集，包含基础外设、传感器、通信协议、云平台对接等 15 个项目


# 1、ESP32 LED灯控制

## 项目功能
实现ESP32板载LED灯的多种控制方式。

## 硬件要求
- ESP32开发板
- USB数据线

## 接线说明
使用ESP32板载LED灯，无需额外接线。

## 使用方法
1. 将代码上传到ESP32开发板
2. 打开串口监视器，波特率设置为115200
3. 通过串口发送指令控制LED灯亮灭

## 关键技术点
- GPIO输出控制
- 串口通信
- 按键输入检测

# 8、ESP32 UDP板间通信

## 项目功能
实现两块ESP32开发板之间基于UDP协议的无线消息收发功能。

## 硬件要求
- 2块ESP32开发板
- 2根USB数据线

## 接线说明
无需额外接线，使用ESP32板载WiFi功能。

## 使用方法
1. 将`UDP-Sender`代码上传到第一块ESP32（发送端）
2. 将`UDP-Receiver`代码上传到第二块ESP32（接收端）
3. 确保两块ESP32连接到同一个WiFi网络
4. 打开两个串口监视器，波特率设置为115200
5. 发送端会每隔1秒向接收端发送一条消息，接收端收到后会在串口打印出来

## 关键技术点
- UDP协议的基本原理与应用。
- ESP32 WiFi STA模式配置。
- 套接字(Socket)编程基础。
- 网络数据传输与解析。


# 9、ESP32与天问51跨平台通信

## 项目功能
实现ESP32与天问51单片机之间的串口通信和引脚通信，完成跨平台LED点灯。

## 硬件要求
- ESP32开发板
- 天问51单片机开发板
- 杜邦线若干
- USB数据线2根

## 接线说明
### 串口通信
| ESP32引脚 | 天问51引脚 |
|----------|-----------|
| GPIO16   | P3.0(RXD) |
| GPIO17   | P3.1(TXD) |
| GND      | GND       |

### 引脚通信
| ESP32引脚 | 天问51引脚 |
|----------|-----------|
| GPIO2    | P1.0      |
| GND      | GND       |

## 使用方法
1. 按照接线说明连接硬件
2. 将ESP32代码上传到ESP32开发板
3. 将天问51代码上传到天问51开发板
4. 打开两个串口监视器，查看通信过程
5. ESP32发送指令，天问51控制LED灯亮灭

## 关键技术点
- UART串口通信
- 跨平台硬件通信
- 数据协议设计
- GPIO输入输出
