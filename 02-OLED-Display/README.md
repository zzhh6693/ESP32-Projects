# ESP32 OLED显示屏驱动

## 项目功能
驱动I2C接口OLED显示屏，实现文字和数据的显示。

## 硬件要求
- ESP32开发板
- 0.96寸I2C OLED显示屏
- 杜邦线4根

## 接线说明
| OLED引脚 | ESP32引脚 |
|---------|----------|
| VCC     | 3.3V     |
| GND     | GND      |
| SDA     | GPIO21   |
| SCL     | GPIO22   |

## 使用方法
1. 按照接线说明连接硬件
2. 将代码上传到ESP32开发板
3. OLED显示屏会显示预设的文字内容

## 关键技术点
- I2C总线通信
- OLED显示屏驱动
- 多界面切换.
